Este proyecto fue construido para las practicas de ingenieria de la universidad central:

Para la creacion de esta plataforma, se usaron las siguientes tecnologias:

- React.js & Next.js
- PWA (native apps, offline access)
- Firebase (authentication, firestore, file hosting)
- Google Analytics
- Netlify (hosting)
- Tailwind CSS (styling)
- Framer Motion (animations)
- [Filepond](https://pqina.nl)
- Google Domains

El codigo base esta en: [GitHub](https://github.com/jramirezs12/reciclon).
